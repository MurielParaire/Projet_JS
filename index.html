Documentation du projet

Présentation du projet

HotHotHot est une application dont le rôle est d’être une interface d’information et de gestion d’un système domotique.

Composition de l'équipe

Notre groupe est composé de 3 personnes :
Bastien FERRARI, Muriel PARAIRE & Antony PINEL.

Outils utilisés

Visual Studio Code

C'était la solution la plus pratique car il était déjà installé sur les ordinateurs de l'IUT et nous l'utilisions souvent pour coder nos projets.

Github

Pour regrouper les différentes parties de code de tous les membres de l'équipe, il était vraiment indispensable pour le versionnage du projet.

Discord

Vu que tout le monde possède déjà ce réseau social, il a été d'une grande aide pour planifier nos objectifs et communiquer les différents changements en continu.

AlwaysData

Une partie du projet demandait à ce que le site soit hébergé sur Internet pour gérer toutes les fonctionnalités de la Progressive Web App ou du Service Worker.
